# Ejr_Nro_13_Catalogo_De_Musica
Ejercicio numero 13 de la guia de ejercicios de Java
# Consigna
Deseamos diseñar un software que mantenga los catálogos de la música, una canción
tiene un autor que pudo componer varias canciones. A la vez un disco está compuesto
por un conjunto de canciones que no necesariamente son de un solo autor.
# Patrones de diseño
* Builder
* Composite
